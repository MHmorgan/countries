# Countries

[REST Countries](https://restcountries.com/) CLI client.

Run the application with the run script:

```sh
# Print help text
./scripts/run --help

# List currencies
./scripts/run currencies

# Show a countries in a region
./scripts/run region europe
```

If you have [nushell](https://www.nushell.sh) installed you can run tests with:

```sh
./scripts/test
```

Otherwise you can run:

```sh
gradle test
```
